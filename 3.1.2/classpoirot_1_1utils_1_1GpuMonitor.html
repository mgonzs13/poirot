Detailed Description

Class for monitoring GPU metrics and energy consumption.

Supports NVIDIA GPUs via nvidia-smi and AMD GPUs via rocm-smi. Provides methods for reading GPU utilization, memory usage, power, and energy.

◆ read_process_energy_uj()

double poirot::utils::GpuMonitor::read_process_energy_uj ( pid_t pid = 0)

Read per-process accumulated GPU energy in microjoules.

This method calculates the energy consumption attributed to a specific process based on its GPU memory usage ratio compared to system-wide GPU usage. Uses time-based integration of power measurements.

Parameters
pidProcess ID to query (default: current process).
Returns
Accumulated energy attributed to the process in microjoules.
